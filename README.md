SalmonttApp1
Descripción
En esta versión se amplía el sistema para gestionar múltiples entidades (unidades operativas y recursos asociados) mediante una interfaz común, polimorfismo y una colección genérica. Además, se incorpora una interfaz gráfica básica que permite ingresar y visualizar registros sin necesidad de usar consola.
Estructura del proyecto
El proyecto está organizado en los siguientes paquetes: 
model/ Contiene las clases e interfaz que representan las entidades del sistema. 
data/ Contiene la clase gestora de entidades y la colección principal. 
ui/ Contiene la interfaz gráfica y la clase principal ejecutable. 
Cómo ejecutar el programa : 
IntelliJ IDEA Abrir el proyecto en IntelliJ IDEA.
Ir al paquete ui. 
Abrir la clase principal: SalmonttGUI (o la clase Main si la nombraste así). 
Ejecutar con el botón Run sobre el método main.
